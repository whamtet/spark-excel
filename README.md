# spark-excel

Quickly spit Spark Dataframes to Excel

## Usage

```scala
spark_excel.core.spitDataframe.invoke("my-file.xlsx", Array("Sheet1", "Another Sheet"), Array(dataframe1, dataframe2, ...))
```
